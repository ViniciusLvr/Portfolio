# Portfólio Pessoal

Portfólio simples feito com React, TypeScript e Vite. Layout inspirado em estética retrofuturista, com cores roxo e verde e uma pegada mais tech minimal.

## O que tem aqui

- Página principal com perfil, apresentação e links de contato
- Lista de projetos com tecnologias usadas e links de demo/código
- Navegação direta para projetos e contato
- Design responsivo para desktop e celular

## Tecnologias

- React
- TypeScript
- Vite
- CSS puro

## Como rodar

```bash
npm install
npm run dev
```

Depois, abra `http://localhost:5173`.

## Build

```bash
npm run build
```

## Onde editar

- `src/components/Header.tsx`: nome, descrição, foto e links de contato
- `src/data/projects.ts`: lista de projetos, descrições e links

## Detalhes úteis

- O projeto é leve e não usa bibliotecas extras além do React
- O visual é focado em contraste e leitura, sem muita complexidade
- Ideal para mostrar trabalho, projetos e contatos de forma direta
